# FunPay-Spooky
Умная автовыдача валюты funpay под спукитайм

## Требования

* **Windows 10/11** — установленный **Node.js LTS** и **npm**.
* **Linux/macOS** — установленный **Node.js LTS** и **npm**.


---

## Настройка

Вся настройка выполняется **в файле `settings.txt`** (расположен в корне проекта, рядом со `start.bat`).

Формат — простые пары `ключ: значение`, по одному параметру на строку. Пример:

```
username: 42_Bot_42
golden_key: ваш_токен
```

---

## Запуск

### Windows

1. Убедитесь, что настроили `settings.txt`.
2. Запустите файл **`start.bat`** двойным кликом или через проводник/терминал:

   ```bat
   start.bat
   ```

### Linux / macOS / *nix

1. Убедитесь, что настроили `settings.txt` (в корне проекта).
2. Выполните установку зависимостей:

   ```bash
   cd path/to/project
   npm i
   ```
3. Запустите приложение:

   ```bash
   node .
   ```

---

## Частые вопросы

**Где лежит `settings.txt`?**  В корне проекта, рядом со `start.bat`.

**Что делает `start.bat`?**  Автоматизирует запуск приложения в Windows (при необходимости может включать установку зависимостей/запуск скриптов). Конкретная логика — в содержимом `start.bat`.

**Не распознаётся команда `npm`**  Установите Node.js LTS с официального сайта и перезапустите терминал/систему, чтобы обновились пути.

## Команды Telegram-бота 

Используйте команды прямо в диалоге с ботом:

* **`чат <текст>`** — бот отправит сообщение с указанным текстом.

  * Пример: `чат Привет! Запуск прошёл успешно.`
* **`баланс`** — бот покажет текущий баланс вашего аккаунта.

> Угловые скобки в примерах означают, что вы подставляете своё значение (вводить их не нужно).
